# Express Full-Stack E-Commerce App

A full-stack e-commerce app with user authentication, products catalog, and shopping cart functionality built with Express.js and SQLite.

## Features
- User authentication (register, login, logout) with session management
- Product catalog with filtering and search
- Shopping cart with add, remove, clear, and view functionality
- Protected routes using middleware
- Clean RESTful API structure with routers and controllers

---

## Technologies
- Node.js & Express.js
- SQLite3 database
- express-session for session management
- bcryptjs for password hashing
- Validator for input validation
